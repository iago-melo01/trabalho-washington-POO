# Projeto Equipe Escola

Sistema escolar desenvolvido em Java como trabalho de Programacao Orientada a Objetos (POO).

## Objetivo

O projeto tem como objetivo criar um sistema para auxiliar na organizacao de informacoes de uma escola, aplicando conceitos de POO como classes, encapsulamento, heranca, polimorfismo e interfaces.

## Tecnologias

- Java
- Git/GitHub
- IDE Java, como Eclipse, IntelliJ ou NetBeans
- Draw.io ou StarUML para diagramas UML
- Google Sites para documentacao da equipe

## Estrutura do Projeto

```text
projeto-equipe-escola/
├── docs/
├── src/
│   ├── exception/
│   ├── main/
│   ├── model/
│   ├── repository/
│   ├── service/
│   └── util/
└── README.md
```

## Organizacao das Pastas

- `docs/`: documentacao do projeto, diagramas, materiais de apoio e arquivos relacionados a apresentacao.
- `src/main/`: ponto de entrada da aplicacao.
- `src/model/`: classes principais do dominio do sistema escolar.
- `src/repository/`: classes responsaveis pelo armazenamento e consulta dos dados.
- `src/service/`: regras de negocio do sistema.
- `src/exception/`: excecoes personalizadas.
- `src/util/`: classes auxiliares e utilitarios.

## Funcionalidades Previstas

- Cadastro e gerenciamento de alunos.
- Cadastro e gerenciamento de professores.
- Organizacao de turmas e disciplinas.
- Registro de dados academicos.
- Consulta de informacoes escolares.

## Documentacao

A documentacao do trabalho sera mantida na pasta `docs/` e tambem no Google Sites da equipe, conforme solicitado na proposta do projeto.
